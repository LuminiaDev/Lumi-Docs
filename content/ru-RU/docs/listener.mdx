# Создание листенера

Как создать listener.

<Callout type="warning">
  Для высокопроизводительных плагинов используйте события без reflection.
</Callout>

## Регистрация событий без reflection

Это зарегистрирует события без reflection:

```java
@Override
public void onEnable() {
    this.getServer().getPluginManager().subscribeEvent(PlayerJoinEvent.class, event -> {
        event.setJoinMessage(TextFormat.DARK_GREEN + event.getPlayer().getName() + " joined the game!");
    }, this);

    this.getServer().getPluginManager().subscribeEvent(PlayerQuitEvent.class, event -> {
        event.setQuitMessage(TextFormat.DARK_GREEN + event.getPlayer().getName() + " left the game!");
    }, this);
}
```

## Регистрация событий с reflection

Это зарегистрирует события с reflection:

```java
public class EventListener implements Listener {

    @EventHandler
    public void onPlayerJoin(PlayerJoinEvent event) {
        event.setJoinMessage(TextFormat.DARK_GREEN + event.getPlayer().getName() + " joined the game!");
    }

    @EventHandler
    public void onPlayerLeave(PlayerQuitEvent event) {
        event.setQuitMessage(TextFormat.DARK_GREEN + event.getPlayer().getName() + " left the game!");
    }
}
```

Затем зарегистрируйте его в вашем плагине:

```java
@Override
public void onEnable() {
    this.getServer().getPluginManager().registerEvents(new EventListener(), this);
}
```
