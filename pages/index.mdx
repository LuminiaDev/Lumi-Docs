# Lumi Documentation

A sunshine between dark Minecraft Bedrock clouds. Lumi is a drop-in replacement for Nukkit focused on performance and modern vanilla features.

## Highlights

- Fast Nukkit fork based on Nukkit-MOT.
- Broad multi-version support for Bedrock protocol ranges.
- More vanilla features: custom blocks, items, enchantments, effects, food, projectiles, and more.
